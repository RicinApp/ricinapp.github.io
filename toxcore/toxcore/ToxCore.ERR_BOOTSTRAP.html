ERR_BOOTSTRAP

Description:

public enum ERR_BOOTSTRAP

Namespace: ToxCore
Package: toxcore

Content:

Enum values:

  • OK - The function returned successfully.
  • NULL - One of the arguments to the function was NULL when it was not expected.
  • BAD_HOST - The address could not be resolved to an IP address, or the IP address passed was invalid.
  • BAD_PORT - The port passed was invalid. The valid port range is (1, 65535).
