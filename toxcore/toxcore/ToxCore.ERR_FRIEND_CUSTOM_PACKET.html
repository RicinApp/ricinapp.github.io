ERR_FRIEND_CUSTOM_PACKET

Description:

public enum ERR_FRIEND_CUSTOM_PACKET

Namespace: ToxCore
Package: toxcore

Content:

Enum values:

  • OK - The function returned successfully.
  • NULL - One of the arguments to the function was NULL when it was not expected.
  • FRIEND_NOT_FOUND - The friend number did not designate a valid friend.
  • FRIEND_NOT_CONNECTED - This client is currently not connected to the friend.
  • INVALID - The first byte of data was not in the specified range for the packet type. This range is 200-254 for lossy, and 160-191 for lossless packets.
  • EMPTY - Attempted to send an empty packet.
  • TOO_LONG - Packet data length exceeded TOX_MAX_CUSTOM_PACKET_SIZE.
  • SENDQ - Packet queue is full.
