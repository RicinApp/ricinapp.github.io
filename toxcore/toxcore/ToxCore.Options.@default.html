public void @default ()

Initialises a Tox_Options object with the default options.

The result of this function is independent of the original options. All values will be overwritten, no values will be read (so it is permissible to pass an uninitialised object).

If options is NULL, this function has no effect.

Parameters:

options

An options object to be filled with default options.
