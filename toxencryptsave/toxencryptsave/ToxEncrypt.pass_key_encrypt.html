pass_key_encrypt

Description:

public bool pass_key_encrypt (uint8[] data, PassKey key, uint8 @out, out ERR_DECRYPTION error)

Encrypt arbitrary with a key produced by tox_derive_key_*. The output array must be at least data_len + TOX_PASS_ENCRYPTION_EXTRA_LENGTH bytes long. key must be TOX_PASS_KEY_LENGTH bytes. If you already have a symmetric key from somewhere besides this module, simply call encrypt_data_symmetric in toxcore/crypto_core directly.

returns true on success
